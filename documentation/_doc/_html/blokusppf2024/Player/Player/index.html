val create_piece_array : Piece.Piece.color -> Piece.Piece.t array

create_piece_array color creates an array of pieces for a given color.

  • color: The color of the pieces to be created.

This function loads piece data from a JSON file located at "./assets/pieces.json". It then creates each piece by converting the coordinates from the JSON data into an array of parts, and initializes each piece with the number of blocks, parts, color, and a default position of (0, 0).

  • parameter color

    The color of the pieces to be created.

  • returns

    An array of pieces with the specified color.
