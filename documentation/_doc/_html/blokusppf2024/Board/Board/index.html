Module Board.Board

type t = {
  1. size : int;
  2. mutable board : Piece.Piece.color array array;
  3. nb_players : int;
  4. mutable placed_pieces : Piece.Piece.t list;
}
val create : int -> t

Creates a new board of the given size.

  • parameter size

    The size of the board to be created.

  • returns

    A new board with the specified size.

val in_matrix : t -> int -> int -> bool

Checks if the given coordinates (x, y) are within the bounds of the matrix.

  • parameter m

    The matrix with a defined size.

  • parameter x

    The x-coordinate to check.

  • parameter y

    The y-coordinate to check.

  • returns

    true if the coordinates are within the bounds of the matrix, false otherwise.

val get : t -> int -> int -> Piece.Piece.color option

Retrieves the value at the specified coordinates (x, y) from the board matrix.

  • parameter m

    The board matrix from which to retrieve the value.

  • parameter x

    The x-coordinate of the value to retrieve.

  • parameter y

    The y-coordinate of the value to retrieve.

  • returns

    Some value if the coordinates are within the bounds of the matrix, None otherwise.

val set : t -> int -> int -> Piece.Piece.color -> unit

Sets the value at the specified coordinates in the matrix if the coordinates are within bounds.

  • parameter matrix

    The matrix in which the value is to be set.

  • parameter x

    The x-coordinate of the position.

  • parameter y

    The y-coordinate of the position.

  • parameter value

    The value to set at the specified position.

  • returns

    Unit. The function modifies the matrix in place.

val find_forbidden_places : t -> Piece.Piece.color -> (int * int) list

find_forbidden_places m col returns a list of positions on the board m that are forbidden for the piece of color col to be placed.

  • parameter m

    the current state of the board

  • parameter col

    the color of the piece

  • returns

    a list of positions that are forbidden for the piece of color col

val find_obligatoy_places : t -> Piece.Piece.color -> (int * int) list

find_obligatory_places m col finds the obligatory places on the board m for the given piece color col.

  • parameter m

    the board matrix

  • parameter col

    the color of the piece

  • returns

    a list of positions that are obligatory for the given piece color, a piece of that color must be placed at one of these positions at least.

val first_piece : Piece.Piece.t -> bool

first_piece p returns a boolean that indicate if the condtions are valid for a first piece.

  • parameter p

    The player whose first piece is to be returned.

  • returns

    a boolean.

val aux_place_piece : t -> Piece.Piece.t -> bool

aux_place_piece m p places the piece p on the board m. It calculates the minimum and maximum coordinates of the piece based on its position and iterates over the range to set the piece's color on the board. The piece is then added to the list of placed pieces in the board m.

  • parameter m

    the board on which the piece is to be placed

  • parameter p

    the piece to be placed on the board

  • returns

    true after the piece is successfully placed

val place_piece : t -> Piece.Piece.t -> bool

place_piece m p places the piece p on the board m.

  • parameter m

    The current state of the board.

  • parameter p

    The piece to be placed on the board.

  • returns

    The updated state of the board after placing the piece.

is_placeable m p checks if the piece p can be placed on the board m.

  • parameter m

    The current state of the board.

  • parameter p

    The piece to be placed on the board.

  • returns

    true if the piece can be placed on the board, false otherwise.

val is_placeable : t -> Piece.Piece.t -> bool
